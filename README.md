### Cmder

- 什么是Cmder?
  - 命令行工具整合WIndows和Linux的命令行
- 怎么使用Windows命令行?
  - 在启动界面 输入cmd
  - 在对应的文件 shift+右键    弹出Context   选择在此打开命令窗口





### 常用的命令 Liunx版本

- cd
  - 进入对应的文件夹
- echo  回声
  - 相当于我们java的的print
  - 两种取变量  win: %PATH% ;  linux : $PATH
- history
  - 记录500条我们输入过的命令
- source
  - Linux加载配置



### 为什么需要配置环境变量:

- 为什么JDK需要配置环境变量
  - 这样你就可以在任意目录 执行对应的可执行文件 比如java.exe
  - 如果不配置环境变量的话 你需要每次都cd到对应可执行文件所在的bin目录
